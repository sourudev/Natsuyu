Google TTS Commands
-lang : เปลี่ยนภาษาบอท 
-langs : แสดงภาษาที่ใช้กับTTSทั้งหมด 
-say : สั่งให้บอทพูดแทนในVoice Channel  
-speed : เปลี่ยนความเร็วในการพูดของบอท (ปรับ normal สำหรับใช้ความเร็วปกติ ,ปรับ slow เพื่อให้พูดช้าลง)

Other TTS Commands
-aeiou : เล่น aeiou (คล้ายๆเกม Moonbase Alpha) ข้อความ TTS ในห้องVoice channel

All TTS Commands
-stop : หยุดบอทและออกจากห้อง

Miscellaneous Commands
-help : แสดงคำสั่งที่ใช้ได้ทั้งหมด
